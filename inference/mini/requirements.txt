# vLLM 安装说明（重要）
# -----------------------------------------------------------------------------
# 本 requirements 故意不列 `vllm`，因为 vllm 的 _C.abi3.so 与 torch 是 ABI 绑定
# 关系，必须保证二者从同一个 CUDA wheel 源装出。一旦 torch 被 -r requirements.txt
# 升级/替换过，vllm 必须重装，否则会出现：
#   ImportError: .../vllm/_C.abi3.so: undefined symbol: _ZN3c104cuda29c10_cuda_...
# （libtorch 的 C++ 符号找不到）
#
# 安装顺序：先装本文件（torch 会被 pin 到 2.9.1），再"单独"装 vllm。
#
# CUDA 12.4：
#   pip install vllm --extra-index-url https://download.pytorch.org/whl/cu124
#
# CUDA 13.0（本项目默认；nightly 渠道，带 glm_ocr 支持）：
#   uv pip install -U vllm --pre \
#     --extra-index-url https://wheels.vllm.ai/nightly/cu130 \
#     --extra-index-url https://download.pytorch.org/whl/cu130 \
#     --index-strategy unsafe-best-match
#
# 升级 torch 或重装 torch 后，务必 `pip uninstall -y vllm` 再按上面重新安装。
# =============================================================================
